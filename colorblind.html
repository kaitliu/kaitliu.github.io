Overview

This project focuses on creating an inclusive web experience for users with color vision deficiency (CVD). The challenge was to design and develop a browser extension that helps colorblind users better distinguish colors on websites, improving their overall browsing experience and accessibility.

Research

Research phase included understanding different types of color vision deficiency, analyzing current accessibility tools, and identifying pain points in existing web experiences for colorblind users.

Design

Design process focused on creating intuitive color adjustment tools, clear visual indicators, and seamless integration with existing web content.

Prototyping

Prototyping involved developing functional prototypes, testing with colorblind users, and iterating based on feedback to ensure the solution effectively addresses accessibility needs.

Reflection

This project highlighted the importance of inclusive design in digital experiences and the need for better accessibility tools in web browsing. The extension serves as a practical solution for colorblind users while raising awareness about accessibility in web design.
